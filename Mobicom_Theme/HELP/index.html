C) CSS Files and Structure - top

Styles

We're using 4 CSS files in this template. we have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

style.css
